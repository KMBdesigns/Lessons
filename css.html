CSS fundamentals

CSS (Cascading Style Sheets) is the language used to describe how HTML should be styled.

CSS uses Selectors to call up elements to style

	.h1

This is the Property

	  { color:

This is the Value

	  blue; }

Putting it together

	.h1 { color: blue; }

The HTML

	<h1>Some blue text</h1>

The CSS

	.h1 { color: blue; }

The result

	Some blue text

Using grid-view to position elements at 100 / 12 = 8.33%

    * { box-sizing: border-box; }
